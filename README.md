# 🍛 Fathima's Biryani House - Food Ordering Website

A modern, responsive food ordering website for Fathima's Biryani House. Built with pure HTML, CSS, and JavaScript - no backend required!

## 🌟 Features

- **Browse Menu**: View all biryani varieties, starters, and desserts
- **Portion Selection**: Choose Half (₹80) or Full (₹160) portions for biryani items
- **Quantity Control**: Add multiple items with quantity selector
- **Shopping Cart**: Review your order with item numbering
- **WhatsApp Integration**: Orders sent directly to WhatsApp
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **Back Navigation**: Easy navigation with back button

## 📱 How to Use

1. Browse the menu and select your favorite items
2. Choose portion size (Half/Full) for biryani items
3. Select quantity and add to cart
4. Review your cart and proceed to checkout
5. Fill in delivery details
6. Click "Order via WhatsApp" to complete your order

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
- LocalStorage for cart management

## 📦 Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR-USERNAME/fathimas-biryani-house.git
```

2. Open `index.html` in your browser

That's it! No installation or build process needed.

## 📝 Customization

### Change Restaurant Name
Search and replace "Fathima's Biryani House" in all HTML files

### Change Phone Number
Edit `checkout.js` line 2:
```javascript
const WHATSAPP_NUMBER = '917019261034';
```

### Add Your Own Images
1. Place images in the `images/` folder
2. Update image paths in `products.js`
3. See `HOW-TO-ADD-IMAGES.txt` for details

### Modify Products
Edit the products array in `products.js` to add/remove/modify menu items
